매일 아침 출근하면 반복되는 작업들이 있습니다. 엑셀 파일에서 데이터를 복사해서 정리하고, 보고서 양식에 맞춰 숫자를 채워 넣고, 특정 조건에 맞는 파일들을 폴더별로 분류하고, 팀원들에게 정해진 시간에 이메일을 보내는 일들. 하나하나는 단순하지만, 쌓이면 하루의 상당 부분을 잡아먹는 업무들이죠.

이런 반복 작업을 **파이썬(Python)**으로 자동화하면 어떨까요? 프로그래밍을 전공하지 않은 분이라도 파이썬의 기초 문법만 익히면 놀라울 정도로 많은 업무를 자동으로 처리할 수 있습니다. 오늘은 실무에서 바로 적용할 수 있는 파이썬 업무 자동화 방법을 구체적으로 살펴보겠습니다.

## 왜 하필 파이썬인가?

업무 자동화 도구는 다양합니다. 매크로, RPA 도구, 심지어 엑셀 VBA도 있죠. 그런데 많은 사람들이 파이썬을 선택하는 데는 분명한 이유가 있습니다.

**첫째, 배우기 쉽습니다.** 파이썬은 영어 문장을 읽는 것처럼 직관적인 문법을 가지고 있습니다. `if`, `for`, `while` 같은 키워드가 그대로 영어 의미와 일치하기 때문에, 코딩 경험이 없는 분도 비교적 빠르게 익힐 수 있습니다.

**둘째, 라이브러리 생태계가 압도적입니다.** 엑셀 처리를 위한 `openpyxl`, 웹 크롤링을 위한 `BeautifulSoup`, 이메일 전송을 위한 `smtplib`, PDF 처리를 위한 `PyPDF2` 등 업무 자동화에 필요한 거의 모든 기능이 이미 라이브러리로 만들어져 있습니다. 바퀴를 새로 발명할 필요가 없다는 뜻이죠.

**셋째, 확장성이 뛰어납니다.** 처음에는 단순한 파일 정리 스크립트로 시작하더라도, 나중에는 데이터 분석, 웹 애플리케이션, 심지어 머신러닝까지 확장할 수 있습니다. 하나의 언어를 배워두면 활용 범위가 계속 넓어지는 셈입니다.

## 실무에서 바로 쓸 수 있는 자동화 사례 5가지

### 1. 엑셀 파일 자동 처리

아마 가장 많은 직장인이 필요로 하는 자동화일 겁니다. 매달 들어오는 매출 데이터를 정리하거나, 여러 시트에 흩어진 정보를 하나로 합치는 작업을 파이썬이 대신해 줍니다.

```python
import openpyxl

# 엑셀 파일 열기
wb = openpyxl.load_workbook('monthly_sales.xlsx')
sheet = wb.active

# 모든 행을 순회하며 매출 합계 계산
total_sales = 0
for row in range(2, sheet.max_row + 1):
    sales_value = sheet.cell(row=row, column=3).value
    if sales_value:
        total_sales += sales_value

# 합계를 새로운 셀에 기록
sheet.cell(row=sheet.max_row + 1, column=3, value=total_sales)
wb.save('monthly_sales_updated.xlsx')
print(f"총 매출: {total_sales:,}원")
```

`pandas` 라이브러리를 함께 사용하면 더 강력합니다. 수백 개의 엑셀 파일을 한 번에 읽어서 조건별로 필터링하고, 피벗 테이블을 만들고, 새로운 파일로 저장하는 것까지 단 몇 줄의 코드로 처리할 수 있습니다.

```python
import pandas as pd
import glob

# 폴더 내 모든 엑셀 파일 합치기
all_files = glob.glob('reports/*.xlsx')
combined_df = pd.concat([pd.read_excel(f) for f in all_files])

# 부서별 매출 요약
summary = combined_df.groupby('부서')['매출'].sum().reset_index()
summary.to_excel('부서별_매출_요약.xlsx', index=False)
```
