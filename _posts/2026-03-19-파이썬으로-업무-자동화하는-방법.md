매일 아침 출근하면 반복되는 작업들이 있습니다. 엑셀 파일 정리, 이메일 발송, 보고서 작성, 폴더 정리… 하나하나는 간단하지만 쌓이면 하루의 상당 부분을 잡아먹죠. "이걸 누가 대신 해주면 좋겠다"라는 생각, 한 번쯤 해보셨을 겁니다.

그 **"누군가"가 바로 파이썬**입니다. 파이썬은 배우기 쉽고, 라이브러리가 풍부하며, 거의 모든 반복 업무를 자동화할 수 있는 강력한 도구입니다. 오늘은 실무에서 바로 써먹을 수 있는 파이썬 업무 자동화 방법을 구체적으로 알려드리겠습니다.

## 왜 하필 파이썬인가?

업무 자동화 도구는 많습니다. 매크로, RPA 툴, 노코드 플랫폼 등 선택지가 다양한데, 그중에서도 파이썬이 특별한 이유가 있습니다.

첫째, **진입 장벽이 낮습니다.** 파이썬은 영어 문장을 읽는 것처럼 직관적인 문법을 가지고 있어서, 프로그래밍 경험이 없는 분도 일주일이면 기본적인 자동화 스크립트를 작성할 수 있습니다.

둘째, **라이브러리 생태계가 압도적입니다.** 엑셀을 다루는 `openpyxl`, 웹 크롤링을 위한 `BeautifulSoup`, 이메일 전송을 위한 `smtplib`, PDF 처리를 위한 `PyPDF2` 등 거의 모든 업무 영역에 맞는 라이브러리가 이미 존재합니다.

셋째, **무료이고 어디서든 실행 가능합니다.** 별도의 라이선스 비용 없이, Windows든 Mac이든 어디서든 동일하게 동작합니다.

## 실무에서 바로 쓰는 자동화 예시 5가지

이론만으로는 와닿지 않으니, 실제로 많은 직장인들이 파이썬으로 자동화하고 있는 대표적인 업무들을 살펴보겠습니다.

### 1. 엑셀 파일 자동 정리 및 병합

여러 부서에서 보내온 엑셀 파일을 하나로 합치는 작업, 수작업으로 하면 30분이지만 파이썬으로는 3초면 끝납니다.

```python
import pandas as pd
import glob

# 폴더 내 모든 엑셀 파일 읽어서 하나로 병합
files = glob.glob("reports/*.xlsx")
all_data = pd.concat([pd.read_excel(f) for f in files])
all_data.to_excel("merged_report.xlsx", index=False)
print("병합 완료!")
```

`pandas` 라이브러리 하나면 엑셀의 거의 모든 작업을 자동화할 수 있습니다. 필터링, 피벗 테이블, 조건부 계산까지 모두 가능하죠.

### 2. 매일 반복되는 이메일 자동 발송

매주 월요일마다 팀원들에게 주간 보고서를 보내야 한다면? 파이썬이 알아서 보내줍니다.

```python
import smtplib
from email.mime.text import MIMEText

def send_email(to_addr, subject, body):
    msg = MIMEText(body)
    msg["Subject"] = subject
    msg["From"] = "me@company.com"
    msg["To"] = to_addr

    with smtplib.SMTP("smtp.company.com", 587) as server:
        server.starttls()
        server.login("me@company.com", "password")
        server.send_message(msg)

send_email("team@company.com", "주간 보고서", "이번 주 보고서를 첨부합니다.")
```

여기에 `schedule` 라이브러리를 결합하면 특정 시간에 자동으로 실행되게 만들 수도 있습니다.

### 3. 웹에서 데이터 자동 수집 (크롤링)

경쟁사 가격 모니터링, 뉴스 키워드 수집, 환율 정보 가져오기 등 웹에서 데이터를 수집하는 작업도 자동화할 수 있습니다.

```python
import requests
from bs4 import BeautifulSoup

url = "https://example.com/news"
response = requests.get(url)
soup = BeautifulSoup(response.text, "html.parser")

headlines = soup.select("h2.news-title")
for h in headlines:
    print(h.text.strip())
```
